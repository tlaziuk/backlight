backlight
===

```
usage: backlight [-h] [-D DIR] [-d | -i] N%

Change the brightness of your displays.

positional arguments:
  N%                    change brightness to given percentage value

optional arguments:
  -h, --help            show this help message and exit
  -D DIR, --directory DIR
                        change directory of display devices
  -d, --dec             decrease value by given percentage value
  -i, --inc             increase value by given percentage value
```
